# ⚡ Alaska Lightning Detection Network Viewer

Interactive dashboard for exploring lightning strike data from the Alaska Lightning Detection Network (ALDN).

## Features
- 🗺️ Interactive heatmap and scatter maps of lightning locations
- 📊 Lightning dynamics: diurnal cycle, monthly patterns, amplitude distributions
- 📈 Time series analysis with 7-day and 30-day rolling averages
- 🔍 Filter by date range, stroke type, and amplitude
- ⬇️ Download filtered data as CSV

## Data Source
Data from the [BLM Alaska Fire Service AICC](https://fire.ak.blm.gov/).  
Updated daily with lightning strikes from the current year.

## How to Use
1. Select a date range in the sidebar
2. Filter by stroke type (CG = Cloud-to-Ground, IC = Intra-Cloud)
3. Adjust amplitude range if needed
4. Explore the four tabs: Map, Dynamics, Temporal Analysis, Raw Data

## To Upload Your Own Data
Upload `aldn_YEAR.csv` or `aldn_YEAR.parquet` (generated by the Colab notebook) to the Space files.
